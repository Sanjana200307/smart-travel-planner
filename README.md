# 🌍 Smart Travel Planner

An AI-powered Smart Travel Planner built with **Python, Flask, HTML, CSS, JavaScript, SQLite, Gemini AI, Geoapify, Google Maps API, and OpenWeather API**.

The application helps users plan trips by providing AI-generated itineraries, tourist attractions, hotels, restaurants, weather forecasts, and budget planning.

---

## ✨ Features

- 🔍 Search destinations
- 🏛 Top tourist attractions
- 🏨 Real-time hotel recommendations
- 🍽 Real-time restaurant recommendations
- 🤖 AI-generated day-wise itinerary
- 🌦 Live weather updates
- 💰 Budget planner
- ❤️ Save favorite trips
- 📍 Google Maps integration
- 📱 Responsive user interface

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/Sanjana200307/smart-travel-planner.git
```

Go to the project folder

```bash
cd smart-travel-planner
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python app.py
```

Open in your browser

```
http://127.0.0.1:5000
```

---

## 🔑 API Keys Required

Create a `.env` file (or use environment variables) and add:

```
GEMINI_API_KEY=YOUR_KEY
GEOAPIFY_API_KEY=YOUR_KEY
OPENWEATHER_API_KEY=YOUR_KEY
GOOGLE_API_KEY=YOUR_KEY
```
